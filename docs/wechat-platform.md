# 微信表情开放平台上架指南

把生成的表情包变成正式上架的微信表情（可被全微信用户搜索下载，可开赞赏）。

## 一键导出素材

结果页点「导出上架素材包」，得到 zip：

| 文件 | 规格 | 平台用途 |
|---|---|---|
| 主图/01-16.png | 240×240 | 表情主图（一套 16 或 24 个） |
| 封面_240.png | 240×240 | 表情封面 |
| 图标_50.png | 50×50 | 聊天面板图标 |
| 横幅_750x400.png | 750×400 | 详情页横幅 |

## 流程

1. **注册**：[sticker.weixin.qq.com](https://sticker.weixin.qq.com)，个人主体可上架（实名）；品牌/IP 建议企业主体（可关联公众号引流）。
2. **创建表情专辑**，逐项上传素材包内容。
3. **填写含义词**：每个表情 1-2 个词，决定搜索曝光——用剧本里的文案词最稳。
4. **审核红线**：无水印二维码联系方式、无商标争议、AI 生成如实勾选、动图 GIF ≤500KB（本工具产物已合规）。
5. 审核 1-7 个工作日；驳回常见原因：画质模糊、含义词不符、文字水印。

## 个人定制 / 品牌 IP

用「AI 策划对话」定制的专属剧本同样可以导出上架——这是把私人梗/品牌口头禅
变成正式微信表情的最短路径。商务定制需求（高分辨率源文件、商用授权书、
代上架）见 README 联系方式。
